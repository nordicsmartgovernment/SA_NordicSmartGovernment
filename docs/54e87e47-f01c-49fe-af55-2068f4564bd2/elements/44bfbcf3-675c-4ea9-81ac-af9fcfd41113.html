Forenklingsprosjektet (SKD + FIN) ()
Regler og ordninger som hindrer automatiskering og digitalisering regelverk med mye skjønn ordninger som krever mye manuell innsats områder som er vankselige å kontorllere regler og forhold som hindrer oversiko goeffektiv bruk av informasjon Eksempler: Slatt og avgift for næringsdrivende: - virksomhetsvurdering og oppstart (kravene for de aller minste) - forskjellene mellom finans- og skatteregnskap - muligheter ofr forenklinger og reduksjon av forkjeller. Eksternt oppdrag (Professor Swenche ved BI, 1. mai) - legge til rette for mer digitaliserest prosesser og informasjonsflyt mellom næringslivet og myndighetene. Standardisert skatterapportering, jf. SIRIUS mv. e.g. kreve system-til-stystemrapporering.
